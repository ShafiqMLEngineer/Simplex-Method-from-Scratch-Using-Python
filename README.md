# Simplex Method from Scratch Using Python

This notebook presents a complete implementation of the Simplex Method from scratch using Python, without relying on optimization libraries such as SciPy.

The objective of this project is to demonstrate how Linear Programming problems can be solved manually through algorithmic steps, including:

* Computing Zj − Cj values
* Selecting the entering variable
* Performing the minimum ratio test
* Identifying the pivot element
* Updating the basis variables
* Executing row operations
* Checking optimality conditions

Rather than treating optimization as a black box, this implementation focuses on understanding the mathematical and computational foundations of the Simplex Algorithm.

This project is intended for students, data scientists, machine learning enthusiasts, and anyone interested in learning how Linear Programming solvers work internally.
